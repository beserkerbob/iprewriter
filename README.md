# iprewriter
A simple script that generates different output formats for a specified IP

```bash
./iprewriter.sh 192.168.1.2
192.168.1.2
3232235778
0300.0250.0001.0002
0xC0.0xA8.0x1.0x2
0xC0.168.0x1.0x2
192.168.1.2
0xC0.168.1.2
0300.168.1.2
192.0xA8.1.2
192.0250.1.2
<snip>

``

It uses hex, octa, decimal and overflow implementations in different formats.
